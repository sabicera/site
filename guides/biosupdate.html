How to update your BIOS

full guide, from start to finish

“please be aware that if anything goes wrong during the update your motherboard will be bricked, unless it has recovery options”

What you will need:

1. USB flash drive (to save the file)

2. Internet connection (to download the file)

3. A reason to update BIOS(new CPU installation, etc)

Avoid if any of this is true:

1. A systems that randomly crashes, or other instabilities

2. Loss of electricity during the update

3. No real benefit of doing an update

How to update the BIOS / UEFI

1. Find your motherboard's model and bios version

2. Go to motherboard's manufacturer website to find the correct bios and download the files. Please read all instructions and check for compatibility issues.

3. Follow any instructions provided from the manufacturer, of how to copy to USB flash drive (filesystem / directories / etc). Usually a simple unzip and copy-paste of the file to the root directory of USB it’s enough.

4. Boot to BIOS setup page pressing the corresponded key (in our case is DEL key)

5. Under the Tools tab you'll find the bios update utility, find your USB drive and select the corresponding bios file. Proceed with the update and wait until it’s finished. The system will reboot or will prompt you a message to reboot manually.

6. All done. You may enter BIOS again to change any settings.

Take extra care for:

All bios settings will be reset to default. Write down any settings you need to carry to the new version (RAID setups, etc).

Make sure CPU is compatible with the version you want to update

If you don't feel comfortable doing so, do NOT update your BIOS. Better find a local PC store or a technician who can do it no problem
