How to install Windows 10 / 11 all editions

from start to finish

In this topic you will get a full guide of how to download windows 10 / 11, create installation USB and boot from it. A USB with at least 8GB or storage is needed. This guide is referred for clean setup of Windows, meaning all data on the drive you will choose has to be DELETED

1 : Get the ISO

Get my latest Windows custom ISO from here or download untouched ISO from Microsoft

2 : Write the ISO to the USB

From this purpose we'll use a free software called Rufus. Run the .exe you just donwloaded

1. Choose you usb flash drive in the "Device" menu

2. Press "SELECT" and choose the .iso file from the folder where you saved it

3. Press "START" to create your Windows installation disk

4. When it says READY you can close the program

3 : Boot from USB

Warning!! Different manufacturers use different keys to enter BOOT MENU

Please read your motherboards manual to find which keys is it

For the majority of desktop PCs the DEL key is for BIOS setup menu, entering this menu you can the BOOT order choosing the drive you just made

4 : Proceed with the setup
