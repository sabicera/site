Various Tips & Tricks for Window users

Use CMD for this one

PowerShell latest version winget.exe install --id Microsoft.Powershell --source winget
Microsoft Edge latest version winget.exe install --id Microsoft.Edge --source winget
Power Plan - High Performance powercfg.exe /setactive 8c5e7fda-e8bf-4a96-9a85-a6e23a8c635c
Turn off screen - Never powercfg.exe /change -monitor-timeout-ac 0
Put to sleep - Never powercfg.exe /x -standby-timeout-ac 0
Hibernation - Off powercfg.exe /hibernate off
Check your BIOS version wmic bios get biosversion
Display your Windows version winver
Power Efficiency Diagnostics Report powercfg/energy
